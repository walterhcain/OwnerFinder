# OwnerFinder
Owner Finder Tool used to find who owns placed barricades and vehicles in Unturned

## How it works
  Mostly an admin tool for finding which player placed a vehicle or barricade. Use the commands to find which player owns it and where the player is located if online.

## Commands
  ```/find``` - Find the current player's coordinates
  ```/find <string>``` - Find the player's coordinates whose name you type as an argument
  ```/checkowner``` - Find the player's name/group/Steam64ID that owns the item that you are looking at

## Permissions
  1. ```checkowner```
  2. ```cain.find```
